У FATE есть три основных инструментария: 

### Fate System Toolkit

14 модулей для кастомизации: магия, технологии, суперсилы, социальные конфликты, мега-данжи, Extras. Гайды по балансу. Используй при создании уникального сеттинга — миксуй для sci-fi, хоррора, супергероев.

Переведён и доступен — [вот тут](/toolkits/stk).

### Fate Adversary Toolkit

Инструменты ГМ для антагонистов: фронты, фракции, угрозы, вожди, иммерсивные кампании вместо статичных монстров. Шаблоны, примеры. Используй для эпичных сюжетов с интригами — когда враги эволюционируют и угрожают миру.

### Fate Accessibility Toolkit

Адаптации правил: альтернативные кубики, нарративные хаки, для слепых (аудио), глухих (визуалы), моторики (голос). Советы по сессиям. Используй в любой группе с особыми нуждами — чтобы все играли на равных.
